# Python Server (FastAPI)

## Залежності

- fastapi - веб-фреймворк
- uvicorn - ASGI сервер
- cryptography - RSA шифрування

## Встановлення

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Сервер буде доступний на `http://127.0.0.1:8080`

## Функціонал

- Генерує RSA ключі (2048 біт) при старті
- Надає публічний ключ клієнтам
- Приймає публічні ключі клієнтів
- Розшифровує повідомлення від клієнтів
- Шифрує відповіді публічним ключем клієнта
