# Go Client

## Залежності

Використовує тільки стандартну бібліотеку Go:
- `crypto/rsa` - RSA шифрування
- `crypto/x509` - кодування ключів
- `encoding/pem` - PEM формат

## Запуск

```bash
go run main.go
```

## Функціонал

- Генерує RSA ключі (2048 біт)
- Отримує публічний ключ сервера
- Відправляє свій публічний ключ серверу
- Шифрує повідомлення публічним ключем сервера
- Розшифровує відповіді своїм приватним ключем
- Консольний інтерфейс для введення повідомлень
