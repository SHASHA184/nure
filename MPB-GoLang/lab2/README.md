## Теоретичні відомості

### 1. Функції

#### Базовий синтаксис
```go
func add(x int, y int) int {
    return x + y
}

// Скорочений синтаксис для параметрів одного типу
func add(x, y int) int {
    return x + y
}
```

#### Вказівники (Pointers)
```go
var p *int          // оголошення вказівника
i := 42
p = &i              // оператор & повертає адресу
fmt.Println(*p)     // оператор * дає доступ до значення
*p = 21             // зміна значення через вказівник
```

### 2. Методи

Метод – це функція із спеціальним аргументом (receiver).

```go
type Vertex struct {
    X, Y float64
}

// Метод з receiver-значенням
func (v Vertex) Abs() float64 {
    return math.Sqrt(v.X*v.X + v.Y*v.Y)
}

// Метод з receiver-вказівником
func (v *Vertex) Scale(f float64) {
    v.X = v.X * f
    v.Y = v.Y * f
}
```

**Коли використовувати receiver-вказівник:**
- Метод повинен змінювати receiver
- Для ефективності (уникнення копіювання великих структур)
- Краще використовувати один тип receiver для всіх методів типу

### 3. Інтерфейси

Інтерфейс визначається набором сигнатур методів.

```go
type Abser interface {
    Abs() float64
}

type MyFloat float64

func (f MyFloat) Abs() float64 {
    if f < 0 {
        return float64(-f)
    }
    return float64(f)
}
```

**Ключові моменти:**
- Інтерфейси реалізуються неявно (немає ключового слова `implements`)
- Порожній інтерфейс `interface{}` може містити значення будь-якого типу
- Інтерфейсне значення nil може викликати panic

```go
// Type switch
switch t := t.(type) {
case bool:
    fmt.Printf("boolean %t\n", t)
case int:
    fmt.Printf("integer %d\n", t)
case *bool:
    fmt.Printf("pointer to boolean %t\n", *t)
}
```

## Корисні поради

### Експортовані та неекспортовані елементи
- **Велика літера** = експортується (public)
- **Мала літера** = не експортується (private)

```go
type Octopus struct {
    Name  string  // експортується
    Color string  // експортується
}

func (o *Octopus) Reset() {  // експортується
    o.Name = ""
}

func (o *Octopus) reset() {  // НЕ експортується
    o.Name = ""
}
```
